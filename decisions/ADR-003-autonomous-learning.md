# ADR-003 - Modelo de Aprendizaje Autónomo

## Contexto
El aprendizaje autónomo requiere que el usuario asuma responsabilidad sobre su progreso. Se debe diseñar el producto para soportar este modelo sin intervención humana.

## Decisión
App Ingreso opera como sistema autónomo donde:
- El usuario elige sus habilidades objetivo
- El sistema adapta la dificultad según rendimiento
- Las métricas de progreso son automáticas y transparentes
- No hay tutor, coach ni soporte humano en el flujo de aprendizaje

El usuario recibe:
- Diagnóstico inicial de nivel
- Ruta de práctica personalizada
- Retroalimentación inmediata por ejercicio
- Visualización clara de su avance

## Consecuencias
- UX debe ser extremadamente clara y autosuficiente
- Onboarding crítico para establecer expectativas correctas
- Producto funciona 24/7 sin intervención
- Métricas deben ser motivacionales, no intimidantes

## Estado
Aceptado
